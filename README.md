# SQLite React Demo

Sample usage SQLite Over HTTP Gateway using Vite + React, deploy on [Surge](https://surge.sh)

Link demo: https://sqlite.surge.sh

## Try it yourself

- Register and Obtain API Key from https://gate.warungds.com `free`
- Clone this repo or copy particular code you want
- Create file `.env` on your root project (see the `sample.env`)
- Run it!
